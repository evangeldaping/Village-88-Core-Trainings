Visual elements that are critical include branding, usability, accessibility and layout. Visual elements that are non-critical include interaction, visual rewards, feedback, and movement. We have to make sure we apply CSS3 to non-critical areas because it will be risky to use CSS3 on critical areas. CSS3 is the cherry on top.
