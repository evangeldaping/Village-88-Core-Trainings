To target the links and remove the underlines from them we can do: #nav-menu li a{ text-decoration: none; } This tells the browser to first locate the element with id nav-menu, then find all the list items inside the unordered lists, then find the links inside those list elements. In general, make your code easier to read by avoiding adding too many classes and ids.
